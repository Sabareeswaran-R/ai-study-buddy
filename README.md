# 🎓 AI-Powered Study Buddy

An intelligent learning assistant built with **Claude AI (Anthropic)**.

## ✨ Features
- 🗣️ **Explain** — Clear explanations of any topic
- 🧒 **ELI5** — Explain Like I'm 5 years old
- ❓ **Quiz** — 5-question MCQ with instant scoring
- 🃏 **Flashcards** — Flip cards with progress tracking
- 🗺️ **Mind Map** — Visual topic breakdown
- 🌐 **12 Languages** — Tamil, Hindi, Arabic, Japanese & more
- 🔊 **Text-to-Speech** — Read aloud in correct language

## 🚀 How to Run
1. Open `AIStudyBuddy.jsx` in Claude.ai artifact
2. Or paste code in any React environment

## 🛠️ Tech Stack
- React JSX + Tailwind CSS
- Anthropic Claude API (`claude-sonnet-4-6`)
- Web Speech API (TTS)
